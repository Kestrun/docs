Kestrun

Kestrun is a hybrid web framework that combines the speed and scalability of ASP.NET Core (Kestrel) with the flexibility and scripting power of PowerShell.
Build web APIs, automation endpoints, and dynamic services using both C# and PowerShell in a single, integrated environment.

Core Capabilities

  • 🚀 Fast, cross-platform web server — Powered by ASP.NET Core (Kestrel)
  • 🐚 Native PowerShell integration — Script-backed routes with dynamic context
  • 🧠 Multi-language script routing — PowerShell, C#, VB.NET, Python, JavaScript, F#
  • 📄 Razor Pages backed by PowerShell — .cshtml + .cshtml.ps1 pairs
  • 📦 Modular architecture — Combine C#, PowerShell, Razor, static files, and more
